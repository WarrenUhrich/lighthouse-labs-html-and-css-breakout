# Lighthouse Labs | HTML and CSS Breakout

## Essential Elements

There are three essential elements required for ANY and all web pages to be valid:

1. `<html>`: root of our document, contains all page contents and information.
2. `<head>`: meta values and external doc references.
3. `<body>`: content belongs here.

## External Doc References

1. CSS documents: `<link>`
2. JavaScript documents `<script>`

<a href=""></a>

<img src="">

<script src=""></script>

<link
  rel="stylesheet"

    RELATIVE PATH:
  href="css/my-styles.css"
  href="./css/my-styles.css"
